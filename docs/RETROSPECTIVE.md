# 项目复盘报告：近期核心重构与优化

**日期**: 2026-04-13  
**项目**: Monad Agentic Payment (daoPayment)

本文档旨在梳理近期项目的四项核心修改内容，包括后端技术栈迁移、网络通信问题修复、前端 UI 视觉升级以及国际化多语言支持。

---

## 1. Python 3.14 / FastAPI 后端迁移

为了提升后端服务的并发处理能力、响应速度以及开发体验，我们将后端架构全面迁移至现代化的 Python 栈。

### 核心改动点：
- **框架升级**：全面采用 **FastAPI** 作为核心 Web 框架，利用其原生支持异步（Asynchronous）和基于 Pydantic 的自动数据验证特性，大幅提升了 API 的稳定性和吞吐量。
- **环境要求**：项目现已要求 **Python 3.14+** 运行环境，通过 `venv` 和 `requirements.txt` 管理依赖，规范了环境部署流程。
- **路由模块化**：在 `app/main.py` 中对路由进行了清晰的拆分，引入了针对钱包（`wallets`）、策略（`policies`）、审计（`audit`）和支付（`payments`）的独立子路由（`APIRouter`），提升了代码的可维护性。
- **CORS 跨域配置**：通过 `CORSMiddleware` 规范了跨域资源共享策略，保障前后端分离架构下的顺畅通信。
- **CLI 工具集成**：引入了基于 Typer 的命令行工具（`app.cli`），方便开发者或管理员在终端直接执行脚本和调试命令。
- **服务器部署**：采用 `uvicorn` 作为高性能 ASGI 服务器进行应用托管。

## 2. 前后端代理网络错误修复

在前后端联调阶段，我们遇到了一些请求代理转发失败（如 `ECONNREFUSED` 或超时）的底层网络问题，现已彻底修复。

### 核心改动点：
- **IPv6 解析规避**：此前 Next.js 的本地代理配置使用了 `localhost`，在部分操作系统中默认会被解析为 IPv6 地址 `::1`，导致无法连接到仅监听 IPv4 的 FastAPI 服务（`127.0.0.1:8000`）。
- **重写规则修正**：修改了 `next.config.mjs` 中的 `rewrites` 规则，将前端向 `/api/:path*` 的请求明确代理至 `http://127.0.0.1:8000/api/:path*`，彻底解决了 Node.js DNS 解析造成的网络层阻断问题。

## 3. 赛博工业风 (Cyber-Industrial) UI 深度优化

为了更契合“Agent-Native”和“区块链”的产品调性，我们对整个前端 UI 进行了深度重构，打造了极具科技感和沉浸感的赛博工业风视觉体验。

### 核心改动点：
- **色彩体系重塑**：在 `app/globals.css` 和 Tailwind 配置中定义了全新的暗黑系主题（Deep Black/Gray 背景），辅以高饱和度的赛博朋克霓虹色（如青翠绿 `#00ffcc` 和亮粉红 `#ff0055`）作为点缀。
- **拟态与材质效果**：
  - 引入了高级毛玻璃质感（Glassmorphism），设计了 `.glass-panel` 和 `.glass-card` 组件类，支持背景模糊（backdrop-blur）和精致的内部阴影（inner shadow）。
  - 添加了底层的 SVG 噪点滤镜（Noise Filter）和 `grid-dao-pattern` 网格背景，增强了界面的工业机械纹理。
- **动态光效与动画**：
  - 设计了 `.neon-trace` 霓虹边框追踪特效、动态流光文本（`.gradient-text`）以及悬浮呼吸动画（`.animate-float`, `.animate-pulse-slow`）。
  - 为组件入场设计了阶梯式显现动画（`.reveal`, `.reveal-1` 至 `.reveal-5`）。
- **移动端适配优化**：针对移动端（<768px）进行了性能妥协，主动关闭或缩短了部分复杂的 CSS 动画延迟，确保在低性能设备上的流畅交互。

## 4. 国际化 (i18n) 双语支持

为满足全球化社区和多元化 DAO 组织的使用需求，系统现已全面接入国际化支持。

### 核心改动点：
- **状态管理与持久化**：基于 `zustand` 结合 `persist` 中间件（`src/i18n/store.ts`），实现了用户语言偏好的本地存储（Local Storage 键值 `daopayment-i18n`），确保用户刷新页面后依然保持所选语言。
- **多语言字典构建**：在 `src/i18n/translations.ts` 中构建了强类型的多语言字典（English `en` / 中文 `zh`），覆盖了从 Landing Page 的营销文案、Wallet/Policy 核心管理模块，到 Audit 审计日志的完整文本映射。
- **语言切换组件**：开发了高度定制化的 `<LanguageSwitcher />` 组件，该组件同样融合了赛博工业风的设计语言（翠绿色边框、Hover 动态滑块特效），支持一键在英文和中文之间无缝切换。
